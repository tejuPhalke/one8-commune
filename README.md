# one8-commune:
One8 Commune Restaurant Website is a modern, fully responsive single-page application (SPA) for a premium restaurant chain. It showcases the restaurant's offerings with an elegant design and smooth user experience.
Key Features:

Multi-page navigation (Home, Menu, Gallery, Contact, Booking) within a single HTML file
Responsive design that adapts seamlessly to mobile, tablet, and desktop devices
Interactive booking system with form validation
Dynamic menu display featuring food and drink categories
Image gallery showcasing multiple restaurant locations
Mobile-friendly hamburger menu with smooth animations
Modern UI/UX with gradient colors, hover effects, and smooth transitions
